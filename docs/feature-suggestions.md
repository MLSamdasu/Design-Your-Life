## 분석 기반

### 현재 앱 상태 요약
- **5탭 구조**: 홈(대시보드), 캘린더(월/주/일), 투두(일정표+체크리스트), 습관/루틴(트래커+시간표), 목표(계층+만다라트)
- **인증**: Google OAuth, 계정 삭제
- **설정**: 다크모드 토글, 계정 관리
- **디자인**: Glassmorphism (MAIN #7C3AED, SUB #EDE9FE)
- **데이터**: Firestore + Hive 로컬 캐시
- **배포**: Firebase Hosting(Web) + Google Play Store(Android)

### 3개 관점 분석 요약

| 관점 | 핵심 발견 |
|---|---|
| 소비자/UX | 현재 앱은 기능 완성도가 높으나, 개인화/알림/소셜 요소가 부재한다. 경쟁사(TickTick, Todoist) 대비 리텐션 유지를 위한 동기부여 장치가 부족하다. |
| 운영/관리 | 모니터링/분석 인프라가 전무하다. 사용자 행동 추적, 에러 수집, 성능 모니터링 없이는 데이터 기반 의사결정이 불가능하다. |
| 비즈니스/재무 | 수익 모델이 미구현 상태다. 프리미엄 구독 체계, 전환 퍼널, 결제 시스템이 필요하다. financial-analysis.md에서 제안한 월 3,900원 Pro 플랜 실행이 시급하다. |

## 권장 구현 순서

### Phase A: 기반 인프라 (MUST, 비즈니스 크리티컬)
```
1. 분석/모니터링 인프라 (다른 모든 기능의 효과 측정 기반)
2. 푸시 알림 시스템 (리텐션의 핵심 동력)
3. 오프라인 모드 강화 (사용 안정성 확보)
4. 데이터 백업/복원 (사용자 신뢰 확보)
```

### Phase B: 수익화 + 플랫폼 확장 (MUST, 성장)
```
5. 프리미엄 구독 시스템 (수익 발생 시작)
6. iOS 지원 (시장 확대)
```

### Phase C: 사용자 경험 강화 (SHOULD)
```
7. 온보딩 개선 (신규 사용자 전환)
8. 고급 통계 대시보드 (프리미엄 가치 강화)
9. 투두/습관 반복 고도화 (핵심 기능 보완)
10. 검색 기능 (파워유저 지원)
11. 소셜 공유 기능 (바이럴 성장)
12. Android 홈 화면 위젯 (DAU 향상)
13. 다국어 지원 (글로벌 진출)
```

### Phase D: 부가 기능 (NICE-TO-HAVE, 선택적)
```
14~20. 사용자 피드백과 데이터 분석 결과에 따라 선택
```
